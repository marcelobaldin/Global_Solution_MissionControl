## Como foi Utilizada

### 1. Estruturacao do Projeto
A IA auxiliou na organizacao da estrutura de arquivos e na definicao das classes do sistema, seguindo o padrao MVC.

### 2. Geracao de Dados Simulados
Os dados de telemetria (dados.csv) e eventos (eventos.csv) foram gerados com auxilio da IA para garantir coerencia com um cenario realista de operacao espacial.

### 3. Implementacao do Codigo
A IA auxiliou na escrita do codigo Python, incluindo as estruturas de dados, regras logicas, algoritmo de regressao linear e interface web.

### 4. Interface Web
O dashboard HTML/CSS/JS com tema espacial foi desenvolvido com auxilio da IA, utilizando Chart.js para graficos.

### 5. Documentacao
README, guia de instalacao e este documento foram elaborados com auxilio da IA.

## Validacao Critica

### O que foi verificado pelo aluno:

1. **Logica das regras:** Cada regra IF/ELIF/ELSE foi revisada para garantir que as condicoes e acoes fazem sentido no contexto de uma missao espacial

2. **Regressao linear:** O algoritmo de minimos quadrados foi verificado manualmente com os dados do CSV para confirmar que os coeficientes estao corretos

3. **Estruturas de dados:** A escolha de cada estrutura foi justificada conforme o problema (fila para alertas FIFO, pilha para eventos LIFO, etc.)

4. **Dados simulados:** Os valores de telemetria foram revisados para garantir coerencia fisica (temperatura marciana, niveis de radiacao, etc.)

5. **Inconsistencia proposital:** A anomalia (geracao solar a meia-noite) foi validada como detectavel pelo sistema

6. **Teste funcional:** O sistema foi executado e testado no navegador, verificando todas as secoes do dashboard
